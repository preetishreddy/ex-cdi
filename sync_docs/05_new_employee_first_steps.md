# New Employee First Steps

Welcome to the team! This guide will help you get started.

## Day 1 Checklist

- [ ] Get laptop and accounts set up by IT
- [ ] Join Slack channels: #onboarding-portal, #dev-team
- [ ] Request access to GitHub repository
- [ ] Request access to Jira (ONBOARD project)
- [ ] Read the Project Overview page

## Day 1-2: Environment Setup

1. Follow the Local Setup Guide to get the project running
2. Ask Marcus or Lisa if you get stuck
3. Make sure you can login to the app locally

## Day 2-3: Explore the Codebase

Key folders to understand:
- `backend/api/` - REST API endpoints
- `backend/services/` - Business logic
- `frontend/src/components/` - React components
- `frontend/src/pages/` - Page layouts

## First Week Goals

- Complete local setup
- Read Technical Architecture doc
- Review open tickets in Jira
- Pick up a small bug fix or documentation task
- Attend team meetings
